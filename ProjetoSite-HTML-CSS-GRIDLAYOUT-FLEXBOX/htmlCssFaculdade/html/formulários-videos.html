Estudando Formulário

Para a criação de um formulário precisamos abrir e fechar a tag "form". Os formulários servem para coletar dados do cliente e fazer algum tipo de processamento no servidor.

Os atributos "name" e "method" são desenvolvidos do lado do servidor, e servem para submeter as informações.

Todos os elementos de um formulário são precedidos da tag "input", e da tag "type", para determinar qual informação será coletada e qual tipo de formulário será utilizado.

Fomulário simples

Os tipos (types) coletam informações como nome, senha, email e url.

O tipo "submit" pega altomaticamente todos os elementos que estão dentro do formulário e submete as informações para o servidor. O atributo "value" é a informação que aparecerá no botão.

Formulário checkbox

O formulário do tipo Checkbox é um formulário de seleção de informações através de caixas de marca e desmarca. Esse tipo de formulário permite que seja selecionado mais de uma opção.

Criar um botão

Outro tipo de botão é o button nele é necessário associar uma programação do lado para que as informações sejam tratadas e submetidas para o servidor.

Escolha de arquivo para enviar

Através do tipo file é possível escolher um arquivo para anexar como informação. Esse tipo habilita uma caixa de seleção para anexar o arquivo.

A programação é feitad lado do servidor para recuperar o arquivo.

Personalizando o botão

É possivel utilizar imagens para criar botões, através do tipo image.

Formulário Radio Button

O formulário que utiliza o tipo radio é parecido com o tipo checkbox, as diferenção que o tipo radio só permite a seleleção de um item e quando selecionado não permite a desmarcação, só a troca de opção.
