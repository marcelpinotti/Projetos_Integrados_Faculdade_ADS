Nesta página serão apresentadas as tags de parágrafo, tabelas, listas e seus tipos, imagens, hiperlinks, entre outras tags.

A tag "p" é a tag que faz a demarcação de um parágrafo, quando é necessário fazer uma quebra de linha utiliza-se a tag "br".

A tag "pre" é uma tag que faz a quebra de linha quando
é dado enter no texto digitado, então essa tag serve para 
inserir quebras ao texto sem precisar usar a tag "br".

Listas Ordenadas e Não ordenadas

As listas ordenadas apresentam marcações numéricas, alfabéticas, etc, antes os itens. Tag de abertura e fechamento da lista "ol", Tag representando o item "il".

As listas não ordenadas apresentam marcadores na frente dos itens. Tad de abertura e fechamento da lista "ul", tag representando o item "il".

Tabelas

As tabelas antigamente eram utilizadas para fazer o layout (organização) das páginas, hoje em dia são utlizadas apenas para tabulação.

As tabelas são abertas e fechadas pela tag "table", as linhas são representadas pela tag "tr" e as colunas são representadas pela tag "td". A tag da linha sempre antecede a tag das colunas.

Imagens (Referências Absolutas e Referências Relativas)

Referência Absoluta é a referência que inclui todo o caminho do arquivo. Usada quando o arquivo referenciado não faz parte da página construída

Referência Relativa é a referência expressa através de um diretório. Usada quando o diretório é conhecido dentro da mesma página.

A tag das imagens é a "img" tendo como atributo o "src". Utiliza-se o atributo "alt" para caso ocorra algum erro no carregamento da imagem apareça uma descrição da imagem. Não deve-se passar o diretório onde a imagem está salva.

Hiperlinks

O Os Hiperlinks são utilizados para acessar um aquivo dentro da própria página ou em outra página

A tag dos hiperlinks é "a", o atributo é o "href" para instanciar o caminho do arquivo e atributo "target" com "_blank" para abrir uma outra página no navegador e não sobrepor a página atual.
